# 📊 Excel Interviewer

An interactive tool built with **Python** and **Streamlit** that allows users to practice Excel interview questions and evaluate their answers automatically.

## 🚀 Features
- ✅ Multiple Excel interview questions
- ✅ Answer evaluation system
- ✅ User-friendly interface via Streamlit
- ✅ Easy to extend with more questions
- ✅ Ready for deployment (Streamlit Cloud / GitHub)


## 🛠️ Tech Stack
-Python 3.9+
- Streamlit
- Pandas
- OpenPyXL

excel_interviewer/

│── README.md          # Documentation

 ⚙️ Installation (Run Locally)

1. **Clone the repository**
   ```bash
   git clone https://github.com/<your-username>/excel_interviewer.git
   cd excel_interviewer
   ```

2. **Create virtual environment (optional but recommended)**
   ```bash
   python -m venv venv
   source venv/bin/activate    # for Mac/Linux
   venv\Scripts\activate       # for Windows
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Run the app**
   ```bash
   streamlit run app.py
   ```

---

## 🌐 Deployment (Streamlit Cloud)

1. Push your code to **GitHub**.
2. Go to [Streamlit Cloud](https://share.streamlit.io/).
3. Click **Create app**.
4. Select your repo → branch → `app.py` file.
5. Click **Deploy** 🚀.
